== Installation ==

1. Upload the entire wfs_googleplay_thumbnail folder to the /wp-content/plugins/ directory or Add New Plugin by upload the wfs_googleplay_thumbnail.zip
2. Activate the plugin through the 'Plugins' menu in WordPress.

You will find 'WFS GPlay Thumbnail' menu in your WordPress admin panel.
